2. Beautiful Subarrays

Count the number of subarrays having exactly m elements.

A beautiful subarray is defined as an array of any length having a specific number of odd elements.  Given an array of integers and a number of odd elements that constitutes beauty, create as many distinct beautiful subarrays as possible.  Distinct means the arrays do not share identical starting and ending indices, though they may share one of the two.

 

Example

arr = [1, 2, 3, 4, 5]

numOdds = 2

 

the following beautiful subarrays can be formed:

[1, 2, 3]

[1, 2, 3, 4]

[2, 3, 4, 5]

[3, 4, 5]

 

Function Description

Complete the function beautifulSubarrays in the editor below.

 

beautifulSubarrays has the following parameter(s):

    int arr[n]:  an array integers

    int numOdds:  the number of odd elements considered beautiful

Returns:

    An integer number of beautiful subarrays.

 

Constraints

  • 1 ≤ n ≤ 2 × 105
  • 1 ≤ arr[i] ≤ 109.
  • The array a consists of distinct positive integers.
  • 0 ≤ numOdds ≤ 2 × 105

 

Input Format for Custom Testing

Input from stdin will be processed as follows and passed to the function.

 

The first line contains an integer n, the size of the array arr.

The next n lines each contain an element arr[i].

The next line contains an integer numOdds, the number of odd elements that constitutes beauty.

Explanation 0

Array arr = [2, 5, 4, 9] has six distinct beautiful subarrays with 1 odd element:

  1. arr[1..1] = [5]
  2. arr[3..3] = [9]
  3. arr[0..1] = [2, 5]
  4. arr[1..2] = [5, 4]
  5. arr[2..3] = [4, 9]
  6. arr[0..2] = [2, 5, 4]
